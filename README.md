# my-artist-list

## Heroku Link
myartistlist.herokuapp.com

## Install Dependencies
npm install

## Project Layout

* HomePage
  - Search for an artist

* ArtistPage
  - See top tracks/upcoming concerts and add artist to favorites list
  - Search for an artist

* FavoritesPage
  - See list of favorite artists with links to artist pages
  - Search for an artist
